# UI_GUIDELINES.md

# RTBrowser UI Guidelines

## UI Philosophy

RTBrowser follows:

* Helium compactness
* Vivaldi efficiency
* Native WinUI rendering
* Instant responsiveness

---

# Layout Rules

## Top Bar Heights

Tab Bar: 30px
Omnibox: 32px
Status Bar: 22px

---

# Navigation Layout

Left:

* Home
* Back
* Forward
* Refresh

Center:

* Compact Omnibox

Right:

* Tiles
* Downloads
* Menu

No:

* profile button
* extensions button
* sidebar

---

# Tabs

Style:

* compact rounded tabs
* minimal padding
* close button on hover only

Tab Width:

* Minimum: 110px
* Preferred: 140px–170px
* Maximum: 220px

Pinned Tabs:

* 36px

---

# Omnibox

Behavior:

* hide https/www
* minimal URL display
* full URL on focus only

Padding:

* Vertical: 4px
* Horizontal: 10px

---

# Typography

Primary Font:
Segoe UI Variable

Sizes:

* Tabs: 12.5px
* Toolbar: 12px
* Menus: 12px
* Status Bar: 11px

---

# Animation Rules

RTBrowser uses Instant UI philosophy.

Allowed:

* tiny hover opacity changes

Avoid:

* bounce animations
* floating motion
* elastic transitions
* heavy acrylic
* blur animations

---

# Status Bar

Height:
22px

Behavior:

* icon-only
* compact spacing
* no labels
* no branding text

Right:

* zoom
* fullscreen

---

# Scrollbars

Use:

* auto-hidden overlay scrollbars

---

# Menus

Use:

* compact native WinUI menus

Avoid:

* oversized menu layouts
* excessive padding
* animated menu systems

---

# Visual Philosophy

RTBrowser should feel:

* dense
* compact
* calm
* silent
* efficient
